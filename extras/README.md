# Vanillagreen Themes

![Vanillagreen theme preview](./assets/vanillagreen-themes.gif)

Matched Ghostty + VS Code-family color themes from the Vanillagreen extras pack.

Preview note: the GIF uses static per-theme frames; shader motion is not shown.

## Install and apply

```bash
vstack add vanillagreen-community/vstack-extras  # or this repo
vstack apply vanillagreen-themes --theme <theme-id> --target ghostty,vscodium
```

## Themes

- `ghibli-serene-nature` — Ghibli Serene Nature
